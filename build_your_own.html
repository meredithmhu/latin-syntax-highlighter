I am pleased to offer support to help users build their own responsive Latin poetry HTML pages, in the style of Meredith's Latin Notebook! Upload a txt file of the poem or Latin writing that you have a translation for and would like an HTML page of. For directions on how to format your txt input file, please refer to the instructions in the following usage doc: USAGE.md.

Note: Currently, annotation generation is not supported for user uploads. All generated HTML pages will not have annotations.
